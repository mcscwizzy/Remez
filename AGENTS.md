# Remez Project Context

## Mission

Remez is a Bible study application focused on multi-layered interpretation:

- Peshat (plain meaning)
- Remez (hint / thematic connection)
- NT parallels (Kesher)
- Hebraic worldview context
- Chiastic structure detection

The goal is structured, layered output that is:

- Theologically grounded
- Contextually aware
- Clear and readable
- API-driven

## Architecture

- Backend: Python (Flask/FastAPI — update if needed)
- Local LLM integration
- REST endpoints for verse analysis
- JSON structured output
- Future: chiastic pattern detection module

## Development Principles

- Keep responses structured
- Modular design
- Clear separation between analysis layers
- Optimize for theological accuracy and traceability

## Current Focus

- API design refinement
- Model prompt structure
- Output formatting schema
